APIs

Fuentes de subtítulos

El addon busca subtítulos en múltiples proveedores y usa AI como respaldo.

OpenSubtitles v3

Proxy gratuito vía strem.io. No necesita API key propia — el proxy maneja la autenticación. Busca por IMDb ID en películas y series. Buena cobertura general.

SubDL

API key gratuita en subdl.com. Busca por IMDb ID. Requiere SUBDL_API_KEY en .env. Series y películas.

Subsource

API key gratuita en subsource.net. Requiere SUBSOURCE_API_KEY en .env. Películas y series.

NVIDIA Riva Translate

Modelo de traducción 4B. API key gratis en build.nvidia.com. Requiere NVIDIA_API_KEY en .env. Traduce inglés a español línea por línea.

Instalación

Paso a paso

1

Configurá tus preferencias

Elegí el modo, provider principal y variante de español en el panel de arriba.

2

Copiá la URL

Hacé clic en "Copiar URL para Stremio" o copiá manualmente el enlace de previsualización.

3

Pegala en Stremio

Andá a Stremio → Addons → botón "Install Addon" → pegá la URL.

4

Disfrutá

Si el addon no encuentra español nativo, traduce desde inglés con AI y te lo muestra igual.
